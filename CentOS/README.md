## 常用操作

|命令|作用|备注|
|:-:|:-:|:-:|
|date|显示日期时间||
|cal|显示日历|`cal \[year]` `cal \[month] \[year]`|
|shutdown|关机|立即关机：`shutdown -h now`|
|reboot|重启||

## 目录文件操作

|命令|作用|备注|
|:-:|:-:|:-:|
|cd|切换目录||
|pwd|显示当前目录||
|mkdir|新建目录||
|rmdir|删除空目录||
|ls|文件与目录管理|-a 显示所有文件|
|cp|复制||
|rm|删除||
|mv|移动||
|cat|第一行开始显示文件内容||
|tac|最后一行开始显示文件内容||
|nl|同时输出行号||
|more|一页一页地显示||
|less|一页一页地显示||
|head|只看前几行||
|tail|只看后面几行||
|od|以二进制方式读取文件内容||
